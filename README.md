# bot-cripto

- App to alert if buy or seel pair cripto selected
- This app get infos of Binance API

## Techs

- Nodejs
- Pkg axios

## Functions

- Alert Price
- RSI Relative Strength Index https://br.tradingview.com/scripts/relativestrengthindex/
- RSI = 100 - (100 / (1 + U/D ))

# Run

```
npm i
```

```
npm start
```
